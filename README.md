# FNB App Academy 2025

## Overview

The **FNB App Academy 2025** is an intensive, hands-on software development program designed to prepare aspiring developers for real-world application development. This experience strengthened my foundation in front-end and back-end development, problem-solving, teamwork, and professional software practices.

This repository documents my learning journey, skills gained, and projects completed during the program.

* **Program:** FNB App Academy
